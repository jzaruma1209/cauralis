## 📖 Descripción

**Cauralis** es una aplicación web moderna construida con **Next.js + TypeScript**. Proyecto en desarrollo activo con enfoque en rendimiento, accesibilidad y una experiencia de usuario premium.

## 🚀 Instalación

```bash
git clone https://github.com/jzaruma1209/cauralis.git
cd cauralis
npm install
npm run dev
```

## ⚙️ Stack Tecnológico

| Tecnología | Uso |
|---|---|
| Next.js 14 | Framework principal |
| TypeScript | Tipado estático |
| React 18 | UI Components |
| CSS3 | Estilos |
